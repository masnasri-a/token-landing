# Public Assets

## Logo Instructions

Please place your `logo.png` file in this directory.

**Required:**
- Filename: `logo.png`
- Recommended size: 512x512px or higher (square format)
- Format: PNG with transparent background recommended
- The logo will be used in:
  - Hero section floating orb (128x128px display)
  - Footer brand logo (48x48px display)
  - Both locations have rotation animation

**Fallback:**
If logo.png is not found, the letter "W" will be displayed as fallback.
